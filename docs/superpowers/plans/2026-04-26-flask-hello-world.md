# Flask Hello World Setup Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Windows 환경에서 Flask Hello World 앱을 로컬 실행 가능한 최소 구성으로 만든다.

**Architecture:** 루트 디렉터리에 Python 가상환경(`.venv`)을 만들고, Flask를 설치한 뒤 단일 엔트리 파일(`app.py`)로 라우트(`/`)를 제공한다. 테스트 프레임워크는 도입하지 않고 수동 실행 검증으로 범위를 제한한다.

**Tech Stack:** Python, venv, Flask

---

### Task 1: Create and activate virtual environment

**Files:**
- Create: `.venv/` (directory generated by venv)

- [ ] **Step 1: Create virtual environment**

Run:
```bash
python -m venv .venv
```
Expected: `.venv` 디렉터리가 생성된다.

- [ ] **Step 2: Activate virtual environment (Windows Bash shell)**

Run:
```bash
source .venv/Scripts/activate
```
Expected: 쉘 프롬프트 앞에 `(.venv)`가 표시된다.

- [ ] **Step 3: Verify Python path points to virtual environment**

Run:
```bash
python -c "import sys; print(sys.prefix)"
```
Expected: 출력 경로가 프로젝트의 `.venv` 경로를 포함한다.

- [ ] **Step 4: Commit**

Run:
```bash
git add .gitignore
git commit -m "chore: initialize python virtual environment"
```
Expected: 가상환경 추적 제외 설정이 있는 경우 커밋된다. (git 저장소가 아니면 이 단계는 생략)

### Task 3: Create minimal Flask app in app.py

**Files:**
- Create: `app.py`

- [ ] **Step 1: Write failing runtime check (server not yet running)**

Run:
```bash
python -c "import urllib.request; urllib.request.urlopen('http://127.0.0.1:5000')"
```
Expected: `ConnectionRefusedError` 또는 연결 실패 에러가 발생한다.

- [ ] **Step 2: Create app.py implementation**

Write `app.py`:
```python
from flask import Flask

app = Flask(__name__)


@app.route("/")
def hello_world():
    return "Hello, World!"


if __name__ == "__main__":
    app.run(debug=True)
```

- [ ] **Step 3: Run Flask app**

Run:
```bash
python app.py
```
Expected: `Running on http://127.0.0.1:5000` 로그가 출력된다.

- [ ] **Step 4: Verify response from root route in separate shell**

Run:
```bash
python -c "import urllib.request; print(urllib.request.urlopen('http://127.0.0.1:5000').read().decode())"
```
Expected: `Hello, World!` 출력.

- [ ] **Step 5: Commit**

Run:
```bash
git add app.py
git commit -m "feat: add flask hello world app entrypoint"
```
Expected: 저장소인 경우 `app.py`가 커밋된다. (git 저장소가 아니면 이 단계는 생략)
